## Getting Started

Install

```
$ git clone git@github.com:KotaTanaka/try-kotlin-react.git
$ cd try-kotlin-react
```

Start - http://localhost:3000

```
$ yarn
$ yarn start
```

Build

```
$ yarn build
```
